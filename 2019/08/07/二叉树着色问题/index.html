二叉树着色问题

发表于 2019-08-07 | 分类于 leetcode

题目

在含有奇数个节点的二叉数中做游戏，首先红蓝玩家依次选择某节点作为阵地的起始点，然后红蓝玩家轮流选择相应阵地的某一节点，将该节点的父子节点可拓展为阵地，循环此步骤，最终阵地多的颜色玩家胜利。

测试用例

失败原因

不会做.

思路

当红色选择节点3时，蓝色最好占据其父子节点，以防扩张，若选择1，则C区域接下来都会被蓝色占据，若选择6，A区域会被蓝色占据，选择7，B区域被占据。我们希望蓝色胜利，当然要选择区域面积最大的，所以max(A,B,C)就是蓝色最大可能的面积，当这个面积大于一半的时候，蓝色胜利。所以这道题其实就是求某一点的左/右子数节点数目的问题。
